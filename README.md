# halodoc-medisend-api

Unofficial python library for consuming Halodoc's Medisend API

## Caveats

- Most of the schema are generated by ChatGPT. Their type annotations might not be accurate.

## Warning

- This library is not officially supported by Halodoc. Use at your own risk.
- Most of the API endpoints are not yet implemented. Feel free to contribute.
- This library is still in development. Expect breaking changes.
- This library is not yet published to PyPI. You can install it by cloning this repository and running `pip install .`
  in the root directory.
